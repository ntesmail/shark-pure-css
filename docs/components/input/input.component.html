Input 输入框

组件说明

  • 1. 需要用户输入表单域内容时。
  • 2. 提供组合型输入框，带搜索的输入框，还可以进行大小选择。

代码演示

基本使用

基本使用，宽度默认100%、高度30px。

规格

input框，高度提供了两种规格，30px(默认)、24px。宽度默认100%，固定宽度提供了三种规格，240px、150px、80px.

前缀和后缀

  • 1.在输入框中添加前缀和后缀。后缀可以增加input-suffix-focus扩展类，input只有在hover和focus状态下显示；
  • 2.禁用状态，除了在input中加上disabled属性，还需要在组件最外层input-affix上增加disabled类。

数字输入框

  • 1.用于输入数字。他的focus状态，需要手动在input-number类上，加上focus类
  • 2.禁用状态，除了在input中加上disabled属性，还需要在组件最外层input-number上增加disabled类。

前置/后置标签

  • 1.用于配置一些固定组合。提供input、input-number、input-affix(带前缀后缀的输入框)与select、btn、input-group-addon的组合；
  • 2.禁用状态，除了在input中加上disabled属性，还需要在组件最外层input-group上增加disabled类。
