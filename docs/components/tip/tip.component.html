tip 气泡提示

组件说明

  • 气泡提示分成以下两类：
  • 1.鼠标移入元素，弹出气泡式的卡片信息提示。
  • 2.点击元素，弹出气泡式的卡片信息提示：提示内容自定义，可以是文字，链接或者按钮等。
