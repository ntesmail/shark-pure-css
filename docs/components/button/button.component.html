Button 按钮

组件说明

  • 1. 可用于各种按钮类操作

代码演示

三种类型

基类是 btn ，扩展类为 btn-primary | btn-secondary | btn-link

三种主题

主题是在 btn-secondary 上进行扩展，默认是次要按钮，对应增加的两个主题类为 btn-secondary-primary | btn-secondary-danger

三种大小

默认是30px高的按钮，btn-sm 对应的高是24px，btn-lg 对应的高是40px

可作为按钮使用的标签或元素

a 标签，button 标签，input 标签（type为 submit | button | reset）
Link
Link

active状态

添加 active 类即可

disabled状态

disabled 状态，注意 a 标签是加 disbled 类，其他是加 disabled 属性

带图标按钮

可在button里增加图标，注意左图标跟右图标需要在i标签上对应增加 btn-icon-left 和 btn-icon-right 类以获得更好的样式

组合按钮

给按钮增加 btn-group 包裹即可实现按钮组

垂直组合按钮

给按钮增加 btn-group-vertical 包裹即可实现垂直按钮组
