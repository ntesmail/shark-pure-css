Progress 加载进度

组件说明

  • 展示操作的当前进度。在操作需要较长时间才能完成时，为用户显示该操作的当前进度和状态。
  • 1. 当一个操作会打断当前界面，或者需要在后台运行，且耗时可能超过2秒时。

代码演示

进度条

一般用于全局页面加载过于缓慢，显示在导航栏底部。添加样式.progress-show-info显示具体进度。

全屏加载

1. 重要内容的加载（例如不加载成功影响后续操），全屏覆盖，加载过程中不可再进行其他操作。
2. 加载时间超过8秒后提供取消加载操作。
3. 为.progress添加样式.progress-show-cancel，显示取消操作按钮。
4. 全屏加载组件不需要另外增加父容器。
