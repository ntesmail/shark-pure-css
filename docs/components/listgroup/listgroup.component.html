List Group 下拉选项

组件说明

  • 1. 可与 select 组件一起使用完成自定义下拉框
  • 2. 下拉选项有两种类型，一种是跟tree结构结合的下拉选项，可以完成单选，多选，全选功能；还有一种是多层级下拉选项

代码演示

普通下拉菜单

宽度需与 select 等宽，最多展示七项，超过七项会出现滚动条，left top width 通过js计算，内联添加
需要添加的类为 group-list 跟 tree 结构结合使用时需要增加一个 tree 类以获得更好的样式
被选中的项在 li 上增加 active 类获得选中样式，disabled 状态时在 li 上增加 disabled 类
