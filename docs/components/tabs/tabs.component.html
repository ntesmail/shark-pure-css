Tabs 选项卡

组件说明

  • 提供平级的区域将大块内容进行收纳和展现，保持界面整洁。
  • Tabs都依赖同一个.tabs类，状态类是tabs-tab-active提供四种类型的选项卡：
  • 1. 卡片型-横向, 需要为tabs添加类型类.tabs-card和.tabs-horizontal
  • 2. 卡片型-纵向, 需要为tabs添加类型类.tabs-card和.tabs-vertical
  • 3. 线型, 需要为tabs添加类型类.tabs-line
  • 4. 块状, 需要为tabs添加类型类.tabs-block
