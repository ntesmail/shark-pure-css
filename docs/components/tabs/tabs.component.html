Tabs 选项卡

组件说明

  • 提供平级的区域将大块内容进行收纳和展现，保持界面整洁，提供四种类型的选项卡。
  • 1. 卡片型-横向
  • 2. 卡片型-纵向
  • 3. 线型
  • 4. 块状
