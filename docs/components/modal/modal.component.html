modal 弹框提示

组件说明

  • 模态对话框
  • 1.需要用户处理事务，又不希望跳转页面以致打断工作流程时，可以使用 Modal 在当前页面正中打开一个浮层，承载相应的操作;
  • 2.当需要一个简洁的确认框询问用户时，可以使用 Modal封装好的确认弹框;
  • 3.当需要一个简洁的提示框提醒用户时，可以使用 Modal封装好的提示弹框。
